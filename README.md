## Подсветка кода

Если нужно выделить слово или фразу внутри строки, используйте одинарные обратные кавычки (`). Например: 

Это `слово` выделено 

Или выделение блока, то тройные кавычки (```). Например:

```
    Здесь может быть
    Ваша реклама
```
